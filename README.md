# ESP32-Multi-Protocol-Gateway-WiFi-6-BLE-LTE-LoRa-GPS-
Gateway IoT multi-protocolo basado en ESP32-C6 con soporte para Wi-Fi 6, BLE, LTE, LoRa y GPS. Incluye diseño PCB personalizado y archivos de fabricación: esquemático PDF, DSN, BRD y Gerbers. Ideal para telemetría, monitoreo remoto, tracking y aplicaciones IoT embebidas.
